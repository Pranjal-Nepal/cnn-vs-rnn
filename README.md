
# Handwritten Digit Classification: CNN vs. RNN Showdown

This deep learning project compares two distinct structural approaches to classifying handwritten digits from the MNIST dataset using PyTorch. The experiment evaluates a spatial Convolutional Neural Network against a sequential Recurrent Neural Network using a two-layer LSTM block. While the CNN processes the entire two-dimensional image plane simultaneously to capture structural geometric patterns like lines and curves, the RNN interprets the static image as a chronological sequence by scanning the pixel grid row-by-row across twenty-eight discrete time steps. Both models were trained using the Adam optimizer and Cross-Entropy loss over ten epochs, easily surpassing the project design targets. The parallel spatial awareness of the CNN yielded a test accuracy of 98.82%, providing a marginal edge over the chronological memory of the LSTM, which achieved an impressive 98.69% accuracy. This comparison demonstrates that while sequential architectures can effectively reconstruct spatial features from sequential streams, spatial models remain the optimal choice for static visual classification tasks.
